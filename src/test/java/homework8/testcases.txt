1. Заголовок: "Ввод в поле Email невалидного адреса эл.почты на странице регистрации в GitHub"
   Приоритет: HIGH
   Предусловия: Открыт сайт на странице регистрации https://github.com/join

   Шаги:
   1) На странице регистрации в поле Email ввести невалидный адреса эл.почты

   Ожидаемый результат:
   - Под полем с электронной почтой выводиться сообщение об ошибке
   - В поле появился значок ошибки

2. Заголовок: "Поиск проектов на сайте GitHub"
   Приоритет: HIGH
   Предусловия: Открыт сайт https://github.com/

   Шаги:
   1) В поле для поиска на сайте GitHub ввести поисковое слово
   2) Нажать Enter

   Ожидаемый результат:
   - В поисковой выдаче находятся названия проектов, которые содержат поисковое слово
